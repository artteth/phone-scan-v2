# Настройка Google Sheets Синхронизации

Это руководство по настройке синхронизации с Google Таблицей для приложения "Сканер рулонов ткани".

## Что даёт синхронизация?

- **Централизованное хранение** - все данные хранятся в вашей Google Таблице
- **Синхронизация между устройствами** - работайте с телефона, планшета, компьютера
- **Резервное копирование** - данные не пропадут при очистке кэша браузера
- **Совместная работа** - несколько пользователей могут работать с одними данными

## Шаг 1: Создание Google Таблицы

1. Откройте https://sheets.google.com
2. Нажмите "+ Создать" → "Google Таблица"
3. Назовите таблицу, например: "Сканер рулонов"
4. Скопируйте ID таблицы из адресной строки:
   ```
   https://docs.google.com/spreadsheets/d/1TsUjce91h44W_PF4dzCqCwGTB_jqhjJxRWBsLiGPmjE/edit
                                          ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                          Это и есть ID таблицы
   ```

## Шаг 2: Создание Google Apps Script

1. В созданной таблице выберите: **Расширения** → **Apps Script**
2. Откроется редактор скриптов
3. Удалите весь код в файле `Code.gs`
4. Скопируйте содержимое файла `cod.gs` из папки проекта
5. Вставьте код в редактор Apps Script

## Шаг 3: Настройка скрипта

В начале файла `cod.gs` найдите и замените:

```javascript
const SPREADSHEET_ID = 'YOUR_SPREADSHEET_ID_HERE'; // Замените на ваш ID
const SHEET_NAME = 'рулоны';
```

На ваши значения:

```javascript
const SPREADSHEET_ID = '1TsUjce91h44W_PF4dzCqCwGTB_jqhjJxRWBsLiGPmjE'; // Ваш ID
const SHEET_NAME = 'рулоны'; // Или любое другое название
```

## Шаг 4: Развёртывание веб-приложения

1. В редакторе Apps Script нажмите **Развернуть** → **Новое развёртывание**
2. Нажмите на шестерёнку и выберите **Веб-приложение**
3. Заполните настройки:
   - **Описание**: `Fabric Inventory API`
   - **Выполнять как**: `Меня` (ваш аккаунт)
   - **Кто имеет доступ**: `Все` (Anyone)
4. Нажмите **Развернуть**
5. Разрешите доступ приложению (нажмите "Review permissions" → Выберите аккаунт → "Allow")
6. Скопируйте **URL веб-приложения**

## Шаг 5: Настройка приложения

1. Откройте приложение "Сканер рулонов ткани"
2. Нажмите на кнопку **⚙️ Настройки** (в правом верхнем углу)
3. Включите переключатель **"Google Таблица Синхронизация"**
4. Вставьте скопированный URL в поле **"Google Apps Script URL"**
5. Нажмите **"🔄 Синхронизировать сейчас"**

## Шаг 6: Проверка работы

1. После синхронизации откройте вашу Google Таблицу
2. Должна появиться вкладка "рулоны" (или другое название)
3. При добавлении/изменении данных в приложении они будут появляться в таблице

## Отключение синхронизации

Чтобы отключить синхронизацию:
1. Откройте настройки (⚙️)
2. Выключите переключатель "Google Таблица Синхронизация"
3. Данные продолжат сохраняться локально

## Важные замечания

### Безопасность
- URL веб-приложения должен храниться в секрете
- Не публикуйте его в открытом доступе
- При компрометации URL перевыпустите развёртывание

### Производительность
- При большом количестве записей (>1000) синхронизация может замедлиться
- Рекомендуется периодически архивировать старые данные

### Ошибки синхронизации
- При отсутствии интернета данные сохраняются локально
- При восстановлении соединения нажмите "Синхронизировать сейчас"
- Все изменения будут отправлены в Google Таблицу

## Устранение проблем

### Ошибка "Server busy"
- Google Apps Script имеет ограничения на одновременные запросы
- Подождите несколько секунд и повторите попытку

### Ошибка "Permission denied"
- Проверьте настройки доступа веб-приложения
- Убедитесь, что выбрано "Кто имеет доступ: Все"

### Данные не синхронизируются
1. Проверьте правильность URL
2. Убедитесь, что синхронизация включена
3. Проверьте консоль браузера на наличие ошибок
4. Попробуйте отключить и снова включить синхронизацию

## API для разработчиков

Google Apps Script поддерживает следующие запросы:

### Получить все заказы
```
GET https://YOUR_SCRIPT_URL/exec?action=getOrders&mode=jsonp&callback=myCallback
```

### Сохранить рулон
```
POST https://YOUR_SCRIPT_URL/exec?mode=tg-api
Content-Type: application/json

{
  "action": "saveRoll",
  "orderId": "2020",
  "rollNumber": 1,
  "factoryLength": 50,
  "measuredLength": 48.5,
  "shrinkage": 3.0
}
```

### Создать заказ
```
POST https://YOUR_SCRIPT_URL/exec?mode=tg-api
Content-Type: application/json

{
  "action": "addOrder",
  "orderId": "2023",
  "totalRolls": 5
}
```
